## نصب

1. پوشه `article-readers-counter` را در مسیر `/wp-content/plugins/` آپلود کنید
2. افزونه را از طریق منوی "افزونه‌ها" در وردپرس فعال کنید
3. به بخش "تنظیمات > شمارنده خوانندگان" بروید و تنظیمات را پیکربندی کنید

## نحوه استفاده

### شورت‌کد ساده:

```php
[readers_count]
```

### شورت‌کد با پارامترها:

```php
[readers_count before_text="در حال مطالعه: " after_text=" خواننده" theme="boxed"]
```

### استفاده در قالب:

```php
<?php echo do_shortcode('[readers_count]'); ?>
```

### پارامترهای در دسترس:

- `before_text`: متن قبل از شمارنده
- `after_text`: متن بعد از شمارنده
- `theme`: قالب نمایش (default, minimal, boxed, rounded, accent)
- `class`: کلاس CSS سفارشی

## پیکربندی

### تنظیمات عمومی:

- درج خودکار در نوشته‌ها
- ثبت تعداد کل بازدیدها

### تنظیمات نمایش:

- متن قبل و بعد از شمارنده
- انتخاب قالب نمایش
- کلاس CSS سفارشی

### تنظیمات پیشرفته:

- فاصله به‌روزرسانی (5 تا 60 ثانیه)
- فاصله پاکسازی (30 تا 300 ثانیه)

## نیازمندی‌ها

- PHP 7.4 یا بالاتر
- وردپرس 5.0 یا بالاتر
- مرورگر مدرن با پشتیبانی از JavaScript

## قالب‌های نمایشی

افزونه دارای چندین قالب نمایشی آماده است:

1. قالب پیش‌فرض

```php
[readers_count theme="default"]
```

2. قالب ساده

```php
[readers_count theme="minimal"]
```

3. قالب جعبه‌ای

```php
[readers_count theme="boxed"]
```

4. قالب گرد

```php
[readers_count theme="rounded"]
```

5. قالب برجسته

```php
[readers_count theme="accent"]
```
